# 智能图片处理工具

一个基于 Web 的智能图片处理工具，支持智能修复、纹理克隆和边缘修复等功能。

## 功能特点

- 智能修复
- 纹理克隆
- 边缘修复
- 支持拖放上传
- 画笔大小调节
- 修复强度控制
- 操作历史撤销
- 实时预览

## 使用方法

访问 [https://ssongjingjie.github.io/image-tools/](https://ssongjingjie.github.io/image-tools/) 即可使用在线版本。

## 本地开发

1. 克隆仓库
2. 使用本地服务器运行项目（如 Live Server）
3. 在浏览器中访问

## 技术栈

- HTML5 Canvas
- JavaScript ES6+
- Web Workers
- CSS3
